Semantic Web Builder (SWB)

Bienvenido a nuestra aplicación estrella
Semantic Web Builder

¿Qué es SWB?

Un conjunto de scripts de Java que permiten construir un portal mediante el uso de páginas creadas con STxT. Aunque a nosotros nos gustaría que fuese usado como "fuente de inspiración" para otros proyectos o desarrollos. Con ese proyecto hemos usado este mismo portal, por lo que estamos muy satisfechos con los resultados.

¿Qué NO es SWB?

SWB no es un editor o GUI para crear portales, ni un entorno de desarrollo web. Para ello hay grandes programas, y no es nuestra intención entrar en competencia con ninguno de ellos.

¿Por qué lo hicisteis?

Realmente hicimos estos scripts en un día; justo cuando empezábamos a hacer esta web. Queríamos algo muy funcional, rápido de hacer, y que nos permitiese tener contenido estático rápidamente publicado. Y resulta que STxT nos aportó la solución. Nos ha gustado tanto que hemos tenido que enseñarlo. Además, es el código fuente con el que hemos construido esta web. ¿Quién más puede presumir de ésto?

¿Qué requerimientos tiene?

Bien, para funcionar es necesario una máquina virtual de Java disponible en el path. El proyecto está creado con Eclipse, por lo que su integración es directa. Para las plantillas hemos usado Apache Velocity.
